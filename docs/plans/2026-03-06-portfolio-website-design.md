# ポートフォリオウェブサイト デザインドキュメント

## 概要

山本浩裕（HY）のフルスタックエンジニア向けポートフォリオサイト。3D/WebGLを活用したニューラルネットワーク空間をヒーローに、開発実績をメインコンテンツとして発信するユニークなウェブサイト。

## コンセプト

**「ニューラルネットワーク空間」** — AI特化フルスタックエンジニアのアイデンティティを3D空間で表現。プロジェクトがノード、技術スタックがエッジとして可視化される。

## 技術スタック

| レイヤー | 技術 | 理由 |
|---------|------|------|
| フレームワーク | Next.js 14 (App Router) | SSGで高速、Vercelとの親和性 |
| 言語 | TypeScript | 型安全性 |
| 3D | Three.js + React Three Fiber (R3F) | Reactで宣言的に3Dを扱える |
| アニメーション | GSAP | スクロール連動アニメーション |
| スタイル | Tailwind CSS | ユーティリティファースト |
| コンテンツ | MDX | Markdownでプロジェクト/ブログ管理 |
| ホスティング | Vercel | CI/CD統合、Edge Network |

## ページ構成

```
/                  → トップ（3Dヒーロー + セクション一覧）
/projects          → 開発実績一覧
/projects/[slug]   → プロジェクト詳細
/blog              → ブログ一覧（将来追加）
/about             → 自己紹介 + スキル + 資格
```

## セクション詳細

### 1. 3Dヒーローセクション

- ダーク背景にネオンブルー/パープルのグラデーションで光るパーティクルノード（50-80個）
- ノードがネットワーク状に接続、ゆっくり回転・脈動
- 中央に「HY」の3Dテキスト（グロウエフェクト付き）
- 下部に「山本浩裕 | AI × Full-Stack Engineer」テキストがフェードイン
- マウス追従でカメラがわずかに動く（パララックス効果）
- ノードホバーで光の波紋エフェクト
- スクロールダウンで3Dシーンが奥に引き、コンテンツセクションへ遷移
- モバイル: パーティクル数削減、シンプルな2D版にフォールバック

### 2. プロジェクトセクション

- カード型グリッド（2-3列、レスポンシブ）
- 各カード: プロジェクト名、技術タグ、サムネイル、概要
- ホバーで浮き上がり + グロウエフェクト
- クリックで詳細ページへ遷移

**初期掲載プロジェクト:**
1. ECサイト SKOSH（Next.js, PostgreSQL, Prisma, Square API）
2. AI動画解析ショート動画自動生成（Python, Gemini API, FFmpeg）
3. 業務支援AIエージェント（Electron, React, NestJS, MongoDB, Azure）
4. 生成AI共通基盤（Go, Kubernetes, OpenShift）
5. Salesforce COMPASS（Apex, Flow, Lightning）
6. 請求書AI解析自動化（Power Automate, AI Builder, OpenAI API）

### 3. プロジェクト詳細ページ

- プロジェクト概要
- 担当範囲・役割
- 技術スタック（タグ表示）
- 成果・学び
- スクリーンショット/デモGIF
- GitHub/外部リンク

### 4. Aboutセクション

- 自己紹介文（キャッチフレーズ: 「AIを製品として組み込めるフルスタックエンジニア」）
- 資格バッジ（AWS SAP, 応用情報, LPIC3, Java Gold, CCNA, Salesforce認定）
- 技術カテゴリ別スキルチャート（言語, FW, AI/LLM, DB, クラウド, DevOps）
- アニメーション付きプログレスバーで経験年数を可視化

## パフォーマンス要件

- Lighthouse Performance: 90+
- 3D: requestAnimationFrame ベースで60fps維持
- ローディング: プログレスバー付きローディング画面
- モバイル: 3Dの軽量版フォールバック
- SSG: 全ページ静的生成
